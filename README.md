### iTerm 2

To get Option-Arrow move by word that the OSX Terminal provides:

Preferences -> Profiles -> Keys

```
Keyboard Shortcut: Option <-
Action: Send Escape Sequence
Esc+: b

Keyboard Shortcut: Option ->
Action: Send Escape Sequence
Esc+: f
```
